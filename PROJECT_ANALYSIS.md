## 📋 Обзор

**Memory Moments** - это full-stack платформа для создания и продажи кастомизированных товаров (футболки, чашки, фотопечать). Проект состоит из двух основных модулей:

1. **T-Shirt Designer** - интерактивный конструктор с 2D и 3D превью
2. **Marketplace** - каталог товаров и админ-панель для управления товарами

## ✨ Функциональность

### 1. Конструктор дизайна (T-Shirt Designer)

#### 2D Canvas (Fabric.js)
✅ **Готово:**
- Загрузка пользовательских изображений (PNG, JPEG, SVG)
- Инструменты трансформации (масштаб, поворот, перемещение)
- Управление слоями (add, remove, reorder)
- Ограничение зоны печати (bounding box)
- Undo/Redo функционал
- Text editing с множеством шрифтов
- Export дизайна

#### 3D Preview (Three.js + React Three Fiber)
✅ **Готово:**
- Real-time рендер 2D дизайна на 3D модель
- Поддержка разных типов товаров (футболка, чашка, поляроид)
- Поворот модели на 360°
- Изменение цвета основы
- Zoom контроль

#### Дополнительные инструменты
✅ **Готово:**
- Фотомаски (circle, oval, heart, square)
- DPI валидатор (предупреждение о низком качестве)
- Line drawing tool
- Smart guides

## ⚠️ Критические проблемы

### 1. Отсутствие интеграции админ-панели и конструктора
**Статус:** 🔴 Критично  
**Описание:** Дизайны, созданные в конструкторе, не сохраняются в БД и не связываются с товарами  
**Решение:** Добавить API для сохранения дизайнов, интегрировать конструктор в форму создания товара

### 2. Слабые UI компоненты админ-панели
**Статус:** 🟡 Важно  
**Описание:** Недостаточный набор компонентов (Select, Checkbox, Textarea с предпросмотром)  
**Решение:** Расширить компоненты из Shadcn/ui, добавить валидацию форм

### 3. Отсутствие управления вариантами товаров
**Статус:** 🟡 Важно  
**Описание:** UI для создания вариантов товаров (размеры, цвета) не реализован  
**Решение:** Добавить динамическую форму для вариантов

### 4. Простой дизайн маркетплейса
**Статус:** 🟢 Низко  
**Описание:** Маркетплейс требует более привлекательного дизайна и функциональности  
**Решение:** Реимплементировать с более красивым UI и фильтрацией

## 🔧 Setup и запуск

### Конструктор
```bash
cd t-shirt-designer-webapp-main
npm install
npm run dev
# Откройте http://localhost:5173
```

### Маркетплейс (API)
```bash
cd marketplace/server
npm install
npm run dev
# API на http://localhost:3001
```

### Маркетплейс (Client)
```bash
cd marketplace/client
npm install
npm run dev
# Клиент на http://localhost:5174
```

### База данных
```bash
cd marketplace/database
mysql -u root -p < schema.sql
mysql -u root -p < seed.sql
```
